Etiquetas HTML

Nombre Propósito Sintaxis Atributos importantes Ejemplo
Title Dar el título de la página <title></title> - <title>Facebook</title>
A Este tag define un link hacia otra página o recurso <a></a>
  • href: Enlace url al recurso
  • target: Dónde se abrirá el recurso
 <a href="www.google.com">Link a Google</a>
Button Este tag define un botón clickeable. Pueden contener imagenes o texto. <button type="button"></button>
  • type: Define el tipo de botón (button, reset, submit)
  • value: Define el valor inicial del botón
 <button type="button">Click Me!</button>
Div Define una división o sección dentro de la página. Obliga hacer un salto de línea <div></div>
  • *
 <div> class="main">Content</div>
Footer Define un pie de página para una sección o la página en sí. <footer></footer>
  • *
 <footer><Content</footer>
Form Se usa para crear un formulario HTML para recibir inputs del usuario <form></form>
  • *
  • action: Lugar donde se enviarán los datos del usuario
  • method: Especifica cuál tipo de método de HTTP utilizar
Headings Usados para proveeer títulos y subtítulos. Generan un bloque nuevo dentro de la página <h4>Texto</h4>
  • *

Este es un heading de prueba
Header Representa un contenedor para para introducir contenido <header></header>
  • *
 <header>Isaac Mena López</header>
Image Permite incluir imágenes en la página <img src="url" alt="nombre">
  • *
  • scr: Url de la imagen, puede ser local o de un servidor
  • alt: Mensaje que se mostrará cuando la imagen no cargue correctamente
  • width: Ancho de la imagen, por defecto se acopla al de la imagen
  • height: Alto de la imagen, por defecto se acopla al de la imagen
Link Define un enlace entre la página y un recurso externo <link rel="relación" type="tipo de recurso" href="url">
  • href: url del recurso
  • rel: relación entre la página y el recurso (estilos, licencia, etc)
  • type: tipo del recurso (javascript, css, icon)
 <link rel="stylesheet" type="text/css" href="url">
Navigation Define un espacio para elementos de navegación <nav>Elementos</nav>
  • *
Link1 Link2 Link2
Paragraph Define un párrafo <p>Texto</p>
  • *

Este es un párrafo de ejemplo
S Tacha el texto para informar que no es correcto <s>Texto</s>
  • *
 Este texto no es correcto
Input Es un espacio donde un usario puede introducir datos <input type="tipo de dato" name="nombre del input" value="valor por defecto" placeholder="valor por defecto">
  • *
  • type: tipo de dato
  • name: nombre del input
  • value: valor por defecto (el usuario tiene que borrarlo para escribir)
  • placeholder: valor por defecto (se quita cuando un usuario escribe)
Style Se usa para proveer estilos a la página internamente <style>Estilos CSS</style>
  • *
  • Type
  • Media
 <style>* {margin: 0px;}</style>

Datalist Provee una lista de datos predefinidos para un input <datalist id="id name"><option value="option value"></article>
  • *
  • id: Id que será utilizado por el input list para desplegar las opciones
Canvas Permite dibujar gráficos mediante scripting usualmente Javascript <canvas>Script</canvas>
  • *
Audio Permite colocar sonido como canciones o streams de audio en la página <audio src="url"></audio>
  • *
  • src: URL del audio
  • autoplay: reproduce automáticamente cuando carga la página
  • controls: provee la interfaz para pausar, adelantar etc
  • loop: deja el audio en un loop infinito de reproducción
Video Permite agregar videos o películas a la pagina de formato MP4, OGG o WebM <video-*></video>
  • *
  • src: URL del audio
  • autoplay: reproduce automáticamente cuando carga la página
  • controls: provee la interfaz para pausar, adelantar etc
  • loop: deja el audio en un loop infinito de reproducción
  • poster: URL de la imagen que se muestra mientras carga el video o este no se ha iniciado
Progress Define una barra de progreso <progress value="valor actual" max="valor máximo"></progress>
  • *
  • value: Valor actual de la barra
  • max: Define el valor máximo que puede tomar la barra, por lo generla es 100
SVG Contenedor para dibujar gráficos SVG <svg width="300" height="250">Figuras</svg>
  • *
  • width: Ancho del canvas
  • height: Alto del canvas

* Incluye los atributos globales como class, id, style, etc y los Atributos de evento como onload, onclick, onresize, etc
